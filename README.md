# Reinforcement Learning in the Gymnastic

This repository documents my learning journey of implementing mainstream reinforcement learning algorithms in the Gym environment, including Deep Q-Network (DQN), Soft Actor-Critic (SAC), Proximal Policy Optimization (PPO), and others.Due to my limited ability, experience, and most importantly, time constraints, some algorithms are not yet fully implemented. This repository serves as a record of my learning process, and I hope it can provide some reference for friends who are also learning reinforcement learning.

这是一个在Gymnastic环境中学习强化学习算法的仓库，记录了我实现Deep Q-Network（DQN） Soft Actor-Critic（SAC） Proximal Policy Optimization（PPO）等主流算法的过程，受限于个人实力与经验以及最重要的时间，部分算法尚未完全实现，仅作为学习过程的记录，希望能为同样在学习强化学习的朋友提供一些参考。
